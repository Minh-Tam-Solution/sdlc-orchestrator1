**NQH-Bot Lesson Applied**: From 679 mocks to zero tolerance

```yaml

  ❌ return { mock: true }, FAKE_DATA, MockService
  ❌ Stub APIs, dummy data, placeholder functions

ALWAYS USE:
  ✅ Real database operations (PostgreSQL + pytest-asyncio)
  ✅ Real API integrations (OPA, MinIO via HTTP)
  ✅ Contract-first validation (OpenAPI 3.0)
  ✅ Factory pattern for test data (real instances)

Enforcement:
  - Pre-commit hook: Scans for banned patterns
  - CI/CD: Automated mock detection (fails build)
  - Code review: Manual verification required
  - Quality gate: Zero tolerance threshold
```
