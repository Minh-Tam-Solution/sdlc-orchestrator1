## Pre-Kickoff Checklist (15-20 min)

Please complete **before Monday 8:30am**:

### 1. Clone & Setup
```bash
git clone --recurse-submodules https://github.com/Minh-Tam-Solution/SDLC-Orchestrator
cd SDLC-Orchestrator
git checkout feature/sop-generator-pilot
```

### 2. Install Dependencies
```bash
# Backend
cd backend && python -m venv venv && source venv/bin/activate
pip install -r requirements.txt

# Frontend
cd ../frontend/web && npm install
```

### 3. Start Services
```bash
# Docker (PostgreSQL, Redis, MinIO, OPA)
docker-compose up -d

# Backend (terminal 1)
cd backend && uvicorn app.main:app --reload --port 8000

# Frontend (terminal 2)
cd frontend/web && npm run dev
```

### 4. Verify Ollama
```bash
curl http://api.nhatquangholding.com/api/tags
# Should return list of available models including qwen3:14b
```

### 5. Smoke Test (TC-001)
```bash
curl -X POST http://localhost:8000/api/sop/generate \
  -H "Content-Type: application/json" \
  -d '{
    "sop_type": "deployment",
    "workflow_description": "Deploy FastAPI application to Kubernetes cluster with zero-downtime. Include health checks, database migrations, and rollback procedure."
  }'
```

**Expected**: SOP generated in <30s with 5 sections.
