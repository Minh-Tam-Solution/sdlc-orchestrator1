## 🚀 Quick Start Prompt

```
Generate 15+ solution ideas for problem:

Problem Statement: [your problem statement from Phase 2]

Methods: SCAMPER + Six Thinking Hats + Brainstorming

Constraints:
- Budget: [amount or range]
- Timeline: [duration]
- Technology: [available tech stack]
- Team: [size and skills]

Evaluate each idea on:
- Feasibility (1-5): How easy to implement
- Impact (1-5): How well it solves problem
- Speed (1-5): How quickly can deliver
- Cost (1-5): Cost effectiveness

Generate:
✅ 15+ diverse solution ideas
✅ Evaluation scores for each
✅ Ranking by combined score
✅ Top 3 recommendations with rationale

Think divergently first, then converge on best ideas
```

## 🎨 Ideation Techniques

### SCAMPER Method
- Substitute: What can be substituted?
- Combine: What can be combined?
- Adapt: What can be adapted from other contexts?
- Modify: What can be modified or magnified?
- Put to other use: New ways to use existing things?
- Eliminate: What can be removed?
- Reverse: What can be reversed or rearranged?

### Six Thinking Hats
- White: Facts and data
- Red: Emotions and feelings
- Black: Risks and problems
- Yellow: Benefits and optimism
- Green: Creativity and new ideas
- Blue: Process and organization
