## 📊 Analysis Framework

### Success Metrics
- Task Completion Rate: >80% target
- Time on Task: Within expected range
- Error Rate: <10% target
- User Satisfaction: >8/10 target
- Feature Adoption: 75-90% target

### Failure Analysis
- Where did users get stuck?
- What caused confusion?
- What features were ignored?
- What workarounds did users create?

### Iteration Priorities
- **High**: Blocks user success, affects >50% users
- **Medium**: Causes frustration, affects 20-50% users
- **Low**: Nice to have, affects <20% users
