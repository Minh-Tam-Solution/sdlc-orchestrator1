## 🎯 Prototype Types

### Paper Prototype (1-2 hours)
- Hand-drawn sketches
- Test concepts quickly
- No coding required
- Best for: Early validation

### Digital Mockup (2-4 hours)
- Figma/Sketch designs
- Interactive prototype
- Looks realistic
- Best for: UI/UX validation

### Code Prototype (1-2 days)
- Functional but limited
- Real technology
- Can test performance
- Best for: Technical validation

### Wizard of Oz (variable)
- Human simulates system
- Tests concept without building
- Quick to set up
- Best for: Complex AI/automation
