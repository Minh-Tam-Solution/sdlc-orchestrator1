## 🤔 What is CodeRabbit?

**Simple explanation**: Like having a senior dev review ALL your PRs 24/7, instantly, for free.

### It does:

✅ Reviews PRs in 2 minutes (vs 2-4 hour wait)  
✅ Finds bugs, security issues, performance problems  
✅ Gives smart suggestions with explanations  
✅ Works in VSCode (optional) and GitHub (automatic)  

### It does NOT:

❌ Auto-merge (you still decide)  
❌ Replace humans (just helps them)  
❌ Make architecture decisions (that's still us)

## ❓ Quick FAQ

**Q: Will it replace human reviewers?**  
A: No. It handles boring stuff (syntax), humans do creative stuff (architecture).

**Q: What if it's wrong?**  
A: You dismiss suggestions. It learns. Human has final say.

**Q: Will it slow me down?**  
A: Goal is 50% faster. If slower after Week 1, we stop.

**Q: What about Zero Mock Policy?**  
A: CTO configured custom rules. 100% enforcement or no-go.

**Q: What about security?**  
A: Using non-sensitive repo. CTO audits tomorrow. Audit fail = stop immediately.

**Q: Do I need to install anything?**  
A: Optional VSCode extension (5 min). GitHub integration is automatic.

**Q: What if I hate it?**  
A: Perfect! Tell us. Your honest feedback decides the outcome.
